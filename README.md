## 🏢 Descripción

**SEPREVIN** es el sitio web oficial de la consultora técnica integral especializada en gestión de riesgos, prevención, seguridad, salud ocupacional y gestión ambiental. Con más de 20 años de experiencia, acompañamos a empresas líderes para que operen con respaldo técnico, menor exposición al riesgo y total claridad frente a inspecciones y auditorías.

## 🌐 Sitio Web

🔗 **URL**: [https://seprevin.vercel.app](https://seprevin.vercel.app)

## 📋 Servicios

- 🛡️ **Seguridad y Salud Ocupacional**: Sistemas de gestión, matrices de riesgos, planes de emergencia
- 🌱 **Gestión Ambiental y SEIA**: Líneas base, evaluación de impacto, seguimiento RCA
- 📋 **Sistemas de Gestión ISO**: ISO 9001, 14001 y 45001, gestión documental completa
- 🗺️ **SIG, Teledetección e IA**: QGIS, ArcGIS, Google Earth Engine, Python
- 🤝 **Asesoría y Acompañamiento**: Soporte técnico preventivo y continuo
- 🎓 **Capacitación**: Programas de formación práctica y actualización normativa

## 🚀 Tecnologías

- **HTML5** - Estructura semántica
- **CSS3** - Diseño responsive con variables CSS
- **JavaScript** - Interactividad y animaciones
- **Google Fonts** - Tipografía Inter y Space Grotesk
- **Unsplash** - Imágenes de alta calidad
- **Formspree** - Gestión de formularios
- **Vercel** - Hosting y despliegue
